#### Lexxca Enabler for nezha

This repository provides Magisk modules for `nezha` devices. Each ZIP changes the reported device profile to enable a different feature set or branding configuration.

## Downloads

### `magisk-module-nezha-lxxca-17U.zip`
Changes the device profile to `Xiaomi 17 Ultra`. Use this package if you want the standard `17 Ultra` profile without the Leica-specific variant.

Download page:
https://github.com/soralis0912/magisk-module-nezha-lxxca/releases/latest

### `magisk-module-nezha-lxxca-17UbyLxxca.zip`
Changes the device profile to `Xiaomi 17 Ultra by Leica`. This package enables the Leica-branded profile and related customized properties.

Download page:
https://github.com/soralis0912/magisk-module-nezha-lxxca/releases/latest

### `magisk-module-nezha-lxxca-Lxxtzphone.zip`
Changes the device profile to `Leitzphone powered by Xiaomi`. Use this package if you want the Leitz Phone style branding and related customized properties.

Download page:
https://github.com/soralis0912/magisk-module-nezha-lxxca/releases/latest
